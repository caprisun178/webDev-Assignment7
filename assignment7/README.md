# Rock, Paper, Scissors React Project

## Setup Instructions

1. Open the project in your terminal.
2. Run `npm run dev`
3. Open the local url link in your browser.

## Project Description

This project uses react effects to let users choose their throw (rock, paper or scissors) vs the computer's choice. Effects is used on the computer throw in order to display the different options while the computer is 'thinking'.

The project uses multiple components:
- PlayerThrow
- ComputerThrow
- ResultDisplay
- ScoreBoard
- ResetButton
